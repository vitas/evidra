Reliability Scoring for

Evidra measures operational reliability across CI pipelines, IaC workflows, and AI agents — by recording evidence, computing behavioral signals, and producing scorecards.

Detect retry loops, artifact drift, and protocol violations. Reliability scoring for CI/CD, Terraform, Kubernetes, and AI agents.

Capabilities

What It Does

A flight recorder for infrastructure automation — observe, measure, score, compare.

◎

Observe

Record every infrastructure operation as signed evidence in an append-only, hash-linked chain.

▤

Measure

Detect 8 behavioral signals: retry loops, artifact drift, protocol violations, blast radius, and more.

★

Score

Compute reliability scorecards with weighted penalties, safety floors, and confidence levels.

⇄

Compare

Compare actors, tools, and scopes side-by-side with workload overlap analysis.

Quick Start

Getting Started

Up and running in under 5 minutes.

# Download latest release (Linux/macOS) curl -fsSL https://github.com/samebits/evidra/releases/latest/download/evidra_$(uname -s | tr '[:upper:]' '[:lower:]')_$(uname -m | sed 's/x86_64/amd64/;s/aarch64/arm64/').tar.gz \ | tar -xz -C /usr/local/bin evidra # Run your first observation evidra run -- kubectl apply -f deploy.yaml # View the scorecard evidra scorecard
# Install via Homebrew brew install samebits/tap/evidra # Run your first observation evidra run -- kubectl apply -f deploy.yaml # View the scorecard evidra scorecard
# Download docker-compose.yml curl -O https://raw.githubusercontent.com/vitas/evidra/main/docker-compose.yml # Set your API key and start export EVIDRA_API_KEY=my-secret-key docker compose up -d # Verify it's running curl http://localhost:8080/healthz
