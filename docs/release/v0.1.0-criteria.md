# v0.1.0 Release Criteria

Evidra v0.1.0 is ready when all of the following are true:

- Argo-first investigation flow works end-to-end.
- Change case files are visible and navigable in UI.
- Correlation search works for app, revision, and external annotation fields.
- Evidence export works deterministically.
- Quickstart succeeds for Docker Compose and kind + Argo CD extension.
- Integration scenario passes on release candidate commit.

## Non-goals
- No governance workflow engine.
- No generic multi-provider ingestion expansion.
- No monitoring/alerting dashboard scope.

## Release checklist
- Contracts and docs aligned with runtime behavior.
- Quickstart validated on clean environment.
- Integration test pass evidence captured.
- Tag created: `v0.1.0`.
