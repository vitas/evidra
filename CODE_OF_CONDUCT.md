# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive.
- Accepting feedback gracefully.
- Focusing on what is best for the project and community.

Examples of unacceptable behavior:

- Harassment, discrimination, or hateful language.
- Trolling, personal attacks, or intimidation.
- Publishing private information without consent.

## Enforcement Responsibilities

Maintainers are responsible for clarifying and enforcing this Code of Conduct.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, and discussions.

## Reporting

Report incidents to the maintainers via private repository contact channels.

## Enforcement

Maintainers may remove, edit, or reject contributions that violate this Code of Conduct.
